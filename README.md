Como atualizar

### Adicionar um novo membro
1. Adicione a foto na pasta `imagens/`
2. Abra `script.js` e localize o array correspondente à geração (`gen2Laura`, `gen2Ildair`, etc.)
3. Insira o novo objeto com `id`, `name`, `img` e `description`
4. Faça o commit no GitHub — o site atualiza automaticamente

Trocar uma foto
Substitua o arquivo em `imagens/` mantendo **exatamente o mesmo nome** (atenção a maiúsculas e minúsculas).
